## SISTEMA DE GESTIÓN DE CITAS MPEDICAS VETERINARIAS 

Sistema que ayuda a una mejor administración de clientes, mascotas con su historial médico y citas médicas de las mascotas, 
utilizando flask, python y SQLite, para el diseño de la interfaz se uso la plantilla con Bootstrap SB Admin 2.

**Dependencias del proyecto**: 
- blinker==1.9.0
- click==8.3.1
- colorama==0.4.6
- Flask==3.1.2
- Flask-SQLAlchemy==3.1.1
- greenlet==3.2.4
- itsdangerous==2.2.0
- Jinja2==3.1.6
- MarkupSafe==3.0.3
- SQLAlchemy==2.0.44
- typing_extensions==4.15.0
- Werkzeug==3.1.3

**Instalación**
1. Clonar este repositorio.
2. Crear el entorno virtual python -m venv env
3. Instalar dependecias pip install -r requirements.txt
4. Ejecutar el archivo created_db.py para crear la BD y tablas.
5. Para iniciar el servidor ejecutar python run.py
6. Pegar en el navegador http://127.0.0.1:5000
